MASINATE PAIGUTUS - DWG/DXF/SVG Rakendus
==========================================

Nõuded: Node.js (https://nodejs.org)

Käivitamine:
  1. Ava terminal selles kaustas
  2. Käivita: node server.mjs
  3. Ava brauseris: http://localhost:3000

Kasutamine:
  - Ava DWG, DXF, SVG või pildifail nupuga "Ava"
  - Joonista kastikesi hiirega klõpsides ja lohistades
  - Sisesta masina nimi
  - Topeltklõps kastil - muuda nime
  - "Vali/Liiguta" režiimis lohista kaste ringi
  - Ekspordi asukohad JSON failina
